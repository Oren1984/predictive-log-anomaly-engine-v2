# Demo Notebook – Predictive Log Anomaly Engine

This directory contains a lightweight demonstration of the **Predictive Log Anomaly Engine** pipeline.

The goal of the demo notebook is to clearly illustrate how the system processes logs and detects anomalous behavioral patterns in event sequences.

The notebook uses a **tiny synthetic dataset** to keep execution fast and stable during demonstrations.

---

# Pipeline Demonstrated

The notebook walks through the full detection pipeline:

Logs  
→ Template Mining  
→ Tokenization  
→ Sliding Window Sequences  
→ Feature Extraction  
→ Anomaly Scoring (Isolation Forest)  
→ Risk Score Normalization  
→ Alert Policy  
→ Visualization

This provides a simplified but accurate representation of how the anomaly engine operates.
